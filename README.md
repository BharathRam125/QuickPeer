# QuickPeer - Cross-Platform File Transfer App

QuickPeer is a cross-platform file transfer application developed as a .NET MAUI Blazor Hybrid app. It allows users to easily and securely transfer files between devices, whether they are running on Android, iOS, or Windows.


## Features

- **Cross-Platform Compatibility:** QuickPeer is built with .NET MAUI and Blazor, ensuring a consistent user experience across Android, iOS, and Windows.

- **User-Friendly Interface:** The app provides an intuitive and user-friendly interface, making file transfers simple and accessible for users of all levels.

- **Secure File Transfer:** QuickPeer uses HTTPS protocol to ensure secure and encrypted file transfers.

- **P2P Connection:** The app establishes peer-to-peer connections, enabling direct communication between devices for faster and more efficient file transfers.

- **File Sharing Without Internet:** QuickPeer allows users to share files without the need for an internet connection, making it an ideal solution for offline file sharing scenarios.


## Prerequisites

To build and run the QuickPeer app, make sure you have the following prerequisites installed on your development machine:

- **.NET 8 SDK**

- **Visual Studio 2022**
